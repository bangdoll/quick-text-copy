單一用途說明：
Quick Text Copy 是一個專注的單一用途生產力工具，設計目標明確且簡潔：快速複製當前網頁的標題和網址到剪貼簿。

核心功能：
• 當使用者點擊擴充功能圖示時，自動讀取當前分頁的標題和網址
• 將資訊格式化為「標題 網址」的統一格式
• 將格式化的文字複製到使用者的系統剪貼簿

我們堅持單一用途原則，不提供其他功能，確保工具的專注性和使用者體驗的一致性。所有操作都在本地完成，不涉及任何網路通訊或資料傳輸。

activeTab 權限說明：
activeTab 權限用於讀取當前活躍分頁的標題和網址資訊。

具體用途：
• 存取 document.title 獲取頁面標題
• 存取 window.location.href 獲取頁面網址
• 僅在使用者主動點擊擴充功能圖示時觸發
• 不會在背景持續監控或存取分頁資訊
• 讀取的資料僅用於格式化和複製，不會儲存或傳輸

此權限確保我們能夠提供核心功能，同時遵循最小權限原則。

scripting 權限說明：
scripting 權限用於在當前分頁中執行複製到剪貼簿的功能。

具體用途：
• 執行 navigator.clipboard.writeText() 將文字寫入剪貼簿
• 在分頁中注入最小化的腳本來處理複製操作
• 僅執行複製相關的腳本，不修改頁面內容或結構
• 不讀取頁面上的其他內容或使用者輸入
• 腳本執行完畢後立即清理，不留下任何痕跡

由於瀏覽器安全限制，此權限是實現剪貼簿功能的必要條件。

遠端程式碼使用說明：
Quick Text Copy 完全不使用任何遠端程式碼或外部資源。

技術實現：
• 所有程式碼都包含在擴充功能套件中
• 不載入外部 JavaScript 檔案或函式庫
• 不連接到任何外部伺服器或 API
• 不使用 CDN 或第三方服務
• 不執行動態載入的程式碼

我們的擴充功能是完全自包含的，確保：
• 完整的離線功能
• 最高的安全性和隱私保護
• 不依賴外部服務的穩定性
• 符合 Chrome Web Store 的安全要求
