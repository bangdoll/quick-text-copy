## 詳細描述

### 🚀 Quick Text Copy - 最簡單的頁面資訊複製工具

厭倦了手動選取網頁標題和複製網址的繁瑣步驟嗎？Quick Text Copy 讓您一鍵搞定！

### ✨ 核心功能

**一鍵複製頁面資訊**
- 點擊擴充功能圖示，立即複製當前頁面的標題和網址
- 自動格式化為「標題 網址」格式，方便分享和記錄
- 無需手動選取文字，節省寶貴時間

**智慧格式化**
- 自動清理多餘空白字元
- 處理過長標題，避免格式混亂
- 支援各種語言的網頁標題

**極速操作體驗**
- 背景執行，無彈出視窗干擾
- 通常在 1 秒內完成複製
- 支援現代 Clipboard API 和傳統備援機制

### 🎯 使用場景

**學術研究與筆記**
- 快速收集參考資料來源
- 建立文獻清單和書籤
- 製作研究報告的引用格式

**工作協作**
- 分享有用的網頁連結給同事
- 建立專案相關資源清單
- 快速記錄會議中提到的網站

**社群分享**
- 在社交媒體分享有趣內容
- 製作推薦清單和資源整理
- 快速回覆朋友的網站詢問

**個人整理**
- 建立個人書籤和筆記
- 整理購物清單和比價資訊
- 收集學習資源和教學材料

### 📋 輸出格式範例

```
蔡正信教練的電腦教學經驗 https://rd.coach/teaching-experience/
GitHub: Let's build from here https://github.com
Google https://www.google.com
Stack Overflow - Where Developers Learn, Share, & Build Careers https://stackoverflow.com
```

### 🔒 隱私與安全

**完全本地處理**
- 不收集任何使用者資料
- 不傳送資訊到外部伺服器
- 所有操作在您的瀏覽器本地完成

**最小權限原則**
- 僅請求必要的 activeTab 和 scripting 權限
- 不存取瀏覽歷史或個人資料
- 透明的權限使用說明

### 🌐 相容性

**廣泛支援**
- 支援 HTTP 和 HTTPS 網站
- 相容各種語言的網頁
- 適用於 Chrome 88 版本以上

**技術特色**
- 使用最新的 Manifest V3 架構
- 現代 Service Worker 技術
- 完整的錯誤處理機制

### 🚀 立即開始使用

1. 安裝擴充功能
2. 前往任意網頁
3. 點擊工具列中的 Quick Text Copy 圖示
4. 到任何地方貼上（Ctrl+V 或 Cmd+V）

就是這麼簡單！讓 Quick Text Copy 成為您提升工作效率的得力助手。
