🚀 Quick Text Copy 擴充功能測試頁面

📋 功能說明

這個 Chrome 擴充功能可以快速複製當前頁面的標題和網址到剪貼簿，格式為：標題 網址

🎯 使用方法

步驟 1: 載入擴充功能

1. 開啟 Chrome 瀏覽器

2. 前往 chrome://extensions/

3. 開啟「開發人員模式」

4. 點擊「載入未封裝項目」

5. 選擇包含 manifest.json 的專案資料夾

步驟 2: 測試功能

1. 在這個測試頁面上點擊瀏覽器工具列的擴充功能按鈕

2. 文字會自動複製到剪貼簿

3. 到任何地方按 Ctrl+V（Windows）或 Cmd+V（Mac）貼上

步驟 3: 查看結果

複製的文字格式應該是：

蔡正信教練的電腦教學經驗 - 漫遊數位 file:///path/to/test-page.html

注意： 實際的網址會是您當前頁面的完整網址。

🌐 不同網站測試

您可以在以下不同類型的網站測試這個擴充功能：

  • GitHub: https://github.com
  • Stack Overflow: https://stackoverflow.com
  • Google: https://www.google.com
  • Wikipedia: https://zh.wikipedia.org

📝 預期輸出格式

根據不同的網站，複製的文字格式會是：

GitHub 範例：

GitHub: Let's build from here https://github.com

Google 範例：

Google https://www.google.com

一般網站範例：

頁面標題 https://example.com/page

您想要的格式範例：

蔡正信教練的電腦教學經驗 https://rd.coach/teaching-experience/

🔧 開發者工具檢查

如果想查看擴充功能的運作情況：

  1. 按 F12 開啟開發者工具
  2. 前往 Console 標籤
  3. 點擊擴充功能按鈕
  4. 查看日誌輸出

✅ 測試檢查清單

  • □ 擴充功能成功載入到 Chrome
  • □ 點擊按鈕後文字複製到剪貼簿
  • □ 複製的文字格式為「標題 網址」
  • □ 在不同網站都能正常運作
  • □ 特殊頁面（chrome://）正確跳過
  • □ 操作快速且無聲

🎉 完成！

如果所有測試都通過，您的 Quick Text Copy 擴充功能就可以正常使用了！

現在您可以在任何網頁上快速複製「標題 網址」格式的文字，方便分享到其他地方。
