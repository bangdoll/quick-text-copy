## 🚀 使用方式

### 安裝步驟
1. 下載 `quick-text-copy-opencc-fixed-v1.0.5.zip`
2. 解壓縮到本地資料夾
3. 在 Chrome 中開啟 `chrome://extensions/`
4. 開啟「開發者模式」
5. 點擊「載入未封裝項目」
6. 選擇解壓後的資料夾

### 使用方法
1. 前往任何包含簡體中文的網站
2. 點擊瀏覽器工具列中的擴充功能圖示
3. 頁面標題和網址會自動複製到剪貼簿
4. **簡體中文會使用內建 OpenCC 核心轉換為繁體中文**

## 🔧 技術細節

### 檔案結構
```
extension-with-opencc-complete/
├── manifest.json                 # 擴充功能配置
├── service-worker.js             # Service Worker
├── opencc-core.js               # 內建 OpenCC 核心
├── content-script-opencc.js     # Content Script
├── privacy-policy.html          # 隱私政策
└── icons/                       # 圖示檔案
```

### 轉換流程
```
1. 用戶點擊擴充功能圖示
2. Service Worker 注入 opencc-core.js
3. Service Worker 注入 content-script-opencc.js
4. Content Script 初始化 OpenCC 核心
5. 取得頁面標題並轉換
6. 複製到剪貼簿
```

### 權限需求
- `activeTab`: 讀取當前分頁資訊
- `scripting`: 注入腳本
