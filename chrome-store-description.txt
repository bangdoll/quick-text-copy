厭倦了手動選取網頁標題和複製網址的繁瑣步驟嗎？Quick Text Copy 讓您一鍵搞定！

✨ 核心功能
• 一鍵複製頁面資訊 - 點擊擴充功能圖示，立即複製當前頁面的標題和網址
• 智慧格式化 - 自動格式化為「標題 網址」格式，方便分享和記錄
• 極速操作體驗 - 背景執行，無彈出視窗干擾，通常在 1 秒內完成複製

🎯 使用場景
• 學術研究與筆記 - 快速收集參考資料來源，建立文獻清單
• 工作協作 - 分享有用的網頁連結給同事，建立專案資源清單
• 社群分享 - 在社交媒體分享有趣內容，製作推薦清單
• 個人整理 - 建立個人書籤和筆記，整理學習資源

🔒 隱私與安全
• 完全本地處理，不收集任何使用者資料
• 最小權限原則，僅請求必要權限
• 使用最新的 Manifest V3 架構

🚀 使用方法
1. 安裝擴充功能
2. 前往任意網頁
3. 點擊工具列中的 Quick Text Copy 圖示
4. 到任何地方貼上（Ctrl+V 或 Cmd+V）
