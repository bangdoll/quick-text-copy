# 🚀 Quick Text Copy v1.0.5 發布說明

## 📅 發布日期
2025年1月13日

## 🎯 主要新功能：Chrome 擴充功能中的簡體轉繁體

v1.0.5 解決了 v1.0.4 中 OpenCC 功能在 Chrome 擴充功能環境中無法使用的問題，現在提供完整的瀏覽器內簡體轉繁體功能！

### ✨ 新功能亮點

#### 🔄 瀏覽器版 OpenCC 轉換器
- **專為 Chrome 擴充功能設計**：無需 Node.js 環境
- **基於字典轉換**：涵蓋常用簡繁體詞彙對照
- **智慧檢測**：自動識別簡體字並轉換
- **高效能**：快速初始化，即時轉換

#### 🖱️ 右鍵選單整合
- **新增選項**：「複製頁面資訊（轉繁體）」
- **一鍵轉換**：右鍵點擊即可複製並轉換
- **智慧通知**：轉換完成後顯示成功通知

#### 📋 雙模式複製
1. **標準模式**：點擊擴充功能圖示 → 複製原文
2. **轉換模式**：右鍵選單 → 複製並轉換為繁體

## 🛠️ 技術實現

### 新增文件
- `opencc-browser.js` - 瀏覽器環境專用轉換器
- `service-worker-with-opencc.js` - 整合 OpenCC 的服務工作者
- `test-opencc-browser.html` - 瀏覽器版本測試頁面

## 🚀 使用方式

### 安裝步驟
1. 下載 `quick-text-copy-with-opencc.zip`
2. 解壓縮到本地資料夾
3. 在 Chrome 中開啟 `chrome://extensions/`
4. 開啟「開發者模式」
5. 點擊「載入未封裝項目」
6. 選擇解壓後的資料夾

### 使用方法
#### 標準複製（不轉換）
- 點擊瀏覽器工具列中的擴充功能圖示

#### 轉換複製（簡體→繁體）
- 在任何頁面右鍵點擊
- 選擇「複製頁面資訊（轉繁體）」

## 🔧 問題解決

### v1.0.4 → v1.0.5 主要修正
- **問題**：OpenCC WASM 無法在 Chrome 擴充功能中運行
- **解決**：創建瀏覽器專用版本，使用字典轉換
- **結果**：簡體轉繁體功能在 Chrome 擴充功能中完全正常

### 常見問題
1. **Q**: 為什麼有些詞彙沒有轉換？
   **A**: 瀏覽器版本使用精選字典，可在 `opencc-browser.js` 中添加更多詞彙

2. **Q**: 如何獲得更準確的轉換？
   **A**: 開發環境可使用 Node.js 版本（`npm run test-opencc`）

3. **Q**: 右鍵選單沒有出現？
   **A**: 確認擴充功能已正確安裝且具有 `contextMenus` 權限
