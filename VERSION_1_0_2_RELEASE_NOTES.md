# 🚀 Quick Text Copy v1.0.2 版本更新

## 📋 版本資訊
- **版本號**: 1.0.2
- **發佈日期**: 2024-08-08
- **更新類型**: 功能增強

## 🆕 新功能

### 🔢 智慧數字標記過濾
- **自動過濾標題開頭的數字標記**
  - 支援圓括號格式：`(3) 標題` → `標題`
  - 支援方括號格式：`[5] 標題` → `標題`
  - 支援大括號格式：`{2} 標題` → `標題`
  - 支援數字點格式：`1. 標題` → `標題`
  - 支援數字括號格式：`3) 標題` → `標題`
  - 支援數字橫線格式：`7 - 標題` → `標題`
  - 支援數字豎線格式：`4 | 標題` → `標題`

- **支援中文數字標記**
  - 中文方括號：`【9】標題` → `標題`
  - 中文圓括號：`〔12〕標題` → `標題`
  - 全形角括號：`＜6＞標題` → `標題`
  - 書名號格式：`《8》標題` → `標題`

### 🛡️ 智慧保護機制
- **防止過度過濾**：過濾後標題太短時保持原樣
- **精確識別**：只處理標題開頭的數字標記
- **保留內容數字**：不影響標題中間或結尾的數字

## 🔄 功能整合

### 完整處理流程
1. **數字標記過濾** - 移除標題開頭的數字標記
2. **簡體轉繁體** - 自動轉換簡體中文
3. **格式化輸出** - 生成 `標題 網址` 格式

### 範例處理
```
輸入: "(3) 【三大模型齐发】OpenAI GPT-OSS - YouTube"
網址: "https://www.youtube.com/watch?v=abc123"

處理步驟:
1. 過濾數字標記: "【三大模型齐发】OpenAI GPT-OSS - YouTube"
2. 簡體轉繁體: "【三大模型齊發】OpenAI GPT-OSS - YouTube"
3. 最終輸出: "【三大模型齊發】OpenAI GPT-OSS - YouTube https://www.youtube.com/watch?v=abc123"
```
